# 🚀 Tracker DevOps + Cybersécurité

Planning personnel de 40 semaines pour devenir DevSecOps.

## Stack
- React + Vite
- Déployé sur GitHub Pages via GitHub Actions

## Lancer en local

```bash
npm install
npm run dev
```

## Déployer
Push sur la branche `main` → déploiement automatique via GitHub Actions.
